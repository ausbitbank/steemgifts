# Steemgifts

Steemgifts is a small website to create an account as a gift and get a cool personalized gift card to go with it. 

install :

Make sure you have an up to date nodejs :

>sudo npm cache clean -f
>sudo npm install -g n
>sudo n stable

then :

> npm i --save
